## 💾 Installation & Setup

```sh
# 📥 Clone the repository
git clone https://github.com/nady4/ai-face-detect.git

# 📂 Move to the project folder
cd ai-face-detect-api

# 📦 Install dependencies
npm install

# 🚀 Run the development server
npm run dev
```
